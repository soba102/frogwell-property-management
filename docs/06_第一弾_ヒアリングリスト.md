# 第一弾案件 ヒアリングリスト（回答用）

**不動産管理業務 AI自動化**（請求書・レポート・消込）

---

## ご案内

拝受したサンプルをもとに、**当社が確認した事実 → 確認したいこと → 回答欄** の順で整理しています。  
可能な項目は **A/B/C から選ぶ** だけで結構です。該当しない場合は備考欄に自由記入してください。

- 氏名・金額はマスキング可
- **スプレッドシートで回答する場合**: [`06_第一弾_ヒアリングリスト_回答用.csv`](./06_第一弾_ヒアリングリスト_回答用.csv) を Google スプレッドシートに取り込み、「回答（先方記入）」列に **選択肢の記号（A/B/C 等）** または短文でご記入ください
