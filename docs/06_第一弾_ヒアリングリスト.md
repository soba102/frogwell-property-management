## ご案内

拝受したサンプルをもとに、**当社が確認した事実 → 確認したいこと → 回答欄** の順で整理しています。

**この Markdown ファイルに直接回答を書いて返送してください。**

- 選択式の質問 → **A / B / C** を記入（複数可の場合はカンマ区切り）
- チェックボックス → `[ ]` を `[x]` に変更（例: `- [x] A`）
- 自由記述 → `>` で囲まれた**記入欄**にそのまま追記
- 氏名・金額はマスキング可

※ スプレッドシート派の方は [`06_第一弾_ヒアリングリスト_回答用.csv`](./06_第一弾_ヒアリングリスト_回答用.csv) も利用可
